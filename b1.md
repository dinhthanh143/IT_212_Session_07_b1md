# BÀI 1: Phân tích & Lựa chọn Prompt

## Đáp án lựa chọn: **B**

### Phân tích tại sao phương án B tối ưu nhất

| Cấu phần | Đánh giá |
|---|---|
| **Input** | Xác định rõ kiểu đầu vào là `String xml`, cấu trúc XML (`<id>`, `<amount>`, `<status>`), Java 17 |
| **Processing** | Chỉ định DOM/SAX (thư viện built-in), kiểm duyệt biên (amount > 0, id không rỗng), ném `InvalidTransactionException` cụ thể, bắt `XMLStreamException`/`ParserConfigurationException` |
| **Output** | `List<TransactionDTO>` với định nghĩa `TransactionDTO` là **Java record** (tận dụng Java 17) |
| **Language** | Vai trò Tech Lead, cấu trúc CoT 3 bước rõ ràng (pseudocode → dry-run → code), không suy diễn sai hướng |
| **Dry-run CoT** | Bắt buộc AI phải mô phỏng lỗi XML trước khi viết code, đảm bảo logic xử lý ngoại lệ được thiết kế đúng |

### Lý do loại trừ A và C

**Phương án A** – Quá mơ hồ: không chỉ định Java version, không nói rõ thư viện XML, không định nghĩa cấu trúc `TransactionDTO`, không phân biệt exception nào cho lỗi nào. AI dễ sinh code thiếu kiểm duyệt biên hoặc dùng sai thư viện.

**Phương án C** – Sai yêu cầu: "song song" (parallel stream) gây overhead không cần thiết; "Spring XML Reader" là thư viện ngoài, mâu thuẫn với yêu cầu dùng thư viện tích hợp sẵn; "bỏ qua dòng lỗi" vi phạm yêu cầu ném `InvalidTransactionException`; "lưu thẳng database" vượt phạm vi; **không có Dry-run CoT** nên AI không được hướng dẫn kiểm tra lỗi biên trước.

### Mã nguồn Java hoàn chỉnh (AI sinh từ prompt B)

```java
import org.w3c.dom.Document;
import org.w3c.dom.Element;
import org.w3c.dom.NodeList;

import javax.xml.parsers.DocumentBuilder;
import javax.xml.parsers.DocumentBuilderFactory;
import javax.xml.parsers.ParserConfigurationException;
import java.io.ByteArrayInputStream;
import java.io.IOException;
import java.nio.charset.StandardCharsets;
import java.util.ArrayList;
import java.util.List;

public record TransactionDTO(String id, double amount, String status) {}

public class TransactionParser {

    public List<TransactionDTO> parseTransactions(String xml) throws InvalidTransactionException {
        List<TransactionDTO> transactions = new ArrayList<>();

        try {
            DocumentBuilderFactory factory = DocumentBuilderFactory.newInstance();
            DocumentBuilder builder = factory.newDocumentBuilder();
            Document document = builder.parse(new ByteArrayInputStream(xml.getBytes(StandardCharsets.UTF_8)));

            NodeList transactionNodes = document.getElementsByTagName("transaction");

            for (int i = 0; i < transactionNodes.getLength(); i++) {
                Element element = (Element) transactionNodes.item(i);

                String id = getTextContent(element, "id");
                String amountStr = getTextContent(element, "amount");
                String status = getTextContent(element, "status");

                if (id == null || id.isBlank()) {
                    throw new InvalidTransactionException(
                        "Transaction id must not be null or empty");
                }

                double amount;
                try {
                    amount = Double.parseDouble(amountStr);
                } catch (NumberFormatException e) {
                    throw new InvalidTransactionException(
                        "Invalid amount value for transaction " + id + ": " + amountStr, e);
                }

                if (amount <= 0) {
                    throw new InvalidTransactionException(
                        "Amount must be greater than 0 for transaction " + id + ", but got: " + amount);
                }

                transactions.add(new TransactionDTO(id, amount, status));
            }

            return transactions;

        } catch (ParserConfigurationException | org.xml.sax.SAXException e) {
            throw new InvalidTransactionException(
                "XML parsing error: malformed XML structure", e);
        } catch (IOException e) {
            throw new InvalidTransactionException(
                "I/O error while reading XML input", e);
        }
    }

    private String getTextContent(Element parent, String tagName) {
        NodeList list = parent.getElementsByTagName(tagName);
        if (list.getLength() > 0) {
            return list.item(0).getTextContent();
        }
        return null;
    }
}
```

**Giải thích Dry-run với XML lỗi (`<id>TX001` thiếu thẻ đóng):** Khi `builder.parse()` gặp thiếu `</id>`, DOM parser ném `SAXException` với message chứa thông tin dòng lỗi → được bắt ở khối `catch (ParserConfigurationException | SAXException e)` và bọc trong `InvalidTransactionException` – người dùng nhận được thông báo lỗi rõ ràng thay vì stack trace kỹ thuật.
